## Usage
### No Overridden Styling
Icons have default styling for SVG styling like fill, stroke, etc. These are defaulted to brand values when no extra props are passed in.
<Playground
  scope={{ CheckIcon }}
  code={`
<CheckIcon size="s" />
<CheckIcon size="m" />
<CheckIcon size="l" />
<CheckIcon size="xl" />
  `}
/>

### Overridden Styling
Fill and color can be passed in as props.
<Playground
  scope={{ CheckIcon }}
  code={`
      <CheckIcon size="m" color="red" fill="purple" />
  `}
/>

### Usage With Other Components
The Icon components can be used inside other components as well, e.g. Button Component
<Playground
  scope={{ CheckIcon, Button, ButtonPreviewContainer }}
  code={`
<ButtonPreviewContainer>
   <Button size="m" icon={<CheckIcon />} >Submit</Button>
   <Button size="m" icon={<CheckIcon />} iconPlacement="left" >Submit</Button>
   <Button size="m" icon={<CheckIcon />} />
</ButtonPreviewContainer>
`}
/>
