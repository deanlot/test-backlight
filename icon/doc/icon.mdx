## Usage
### No Overridden Styling
Icons have default styling for SVG styling like fill, stroke, etc. These are defaulted to brand values when no extra props are passed in.
<Playground
  scope={{ Icon }}
  code={`
    <Icon icon="home"  />
  `}
/>

### Overridden Styling
Styling can be overridden if required as well.
<Playground
  scope={{ Icon }}
  code={`
    <Icon icon="home"  fill="#F0BD6B" stroke="#D17F13" />
  `}
/>
