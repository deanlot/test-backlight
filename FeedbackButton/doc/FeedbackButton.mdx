# FeedbackButton
<Description of={FeedbackButton} />

### Success state
<p>
In the below usage, the busy indicator will be displayed while the promise is being executed, once completed, te button will display the success icon briefly to the user.
</p>
<Playground
  scope={{ FeedbackButton }}
  code={`
  <FeedbackButton
    variant="primary"
    inferBusy
    onClick={() => new Promise((resolve) => setTimeout(resolve, 3000))}
  >
    Click Me
  </FeedbackButton>
  `}
/>

### Failure state
<p>
  If the promise passed in fails the button will shake as a visual cue to the end user.
</p>
__Note: the exception is caught and thrown back up to the implementation__
<Playground
  scope={{ FeedbackButton }}
  code={`
  <FeedbackButton
    variant="primary"
    inferBusy
    onClick={() => new Promise((resolve, reject) => setTimeout(reject, 2000))}
  >
    Click Me
  </FeedbackButton>
  `}
/>
